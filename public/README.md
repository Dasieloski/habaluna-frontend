# Carpeta Public

Esta carpeta contiene archivos estáticos que se sirven directamente.

## Logo

Coloca tu archivo de logo aquí con uno de estos nombres:

- `logo.svg` (recomendado para mejor calidad)
- `logo.png` (alternativa si no tienes SVG)

El componente Logo buscará primero `logo.svg` y si no existe, intentará cargar `logo.png`.

### Ubicación del archivo:

```
frontend/public/logo.svg
```

o

```
frontend/public/logo.png
```
